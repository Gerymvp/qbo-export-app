Política de Privacidad

En QBO-EXPORT-APP, la privacidad de sus datos financieros es nuestra prioridad.

1. Datos que Recolectamos

  • Credenciales de Acceso: Gestionadas de forma segura a través de Supabase Auth.
  • Tokens de QuickBooks: Almacenamos temporalmente los Access Tokens y Realm IDs necesarios para la comunicación con la API de Intuit.
  • Datos de Facturación: Procesamos sus archivos XML para extraer datos de proveedores y montos.

2. Uso de la Información

La información recolectada se utiliza exclusivamente para la creación de "Bills" (Facturas de proveedores) en su cuenta de QuickBooks. No vendemos ni compartimos sus datos con terceros ajenos al flujo técnico del sistema.

3. Almacenamiento Seguro

Utilizamos Supabase con políticas de seguridad de nivel de fila (RLS) para asegurar que solo usted pueda acceder a sus tokens y datos de facturas pendientes.

4. Control del Usuario

Usted puede revocar el acceso a su cuenta de QuickBooks en cualquier momento desde el panel de configuración de la aplicación o desde su portal de Intuit.
